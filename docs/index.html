phantom

Pure AMOLED black meets electric purple. A Discord theme engineered for those who demand absolute darkness with surgical precision.

Download Theme
⚫

Pure AMOLED

Absolute pitch black (#000000) backgrounds. Not dark gray, not charcoal. True black that turns off pixels on OLED displays, saving battery and looking immaculate.

⚡

Electric Purple

Surgical accent color deployment. Purple is used sparingly and precisely—interactive states, mentions, and critical UI elements only. No purple fatigue.

🎯

Zero Bloat

Single CSS import. No JavaScript dependencies. No external assets. The entire theme is leaner than most favicons, loading instantly every time.

🔧

Native Integration

Built to work seamlessly with BetterDiscord, Vencord, and Replugged. Uses CSS variables natively so other plugins can theme on top without conflicts.

Quick Import

One line. Zero configuration. Instant transformation.

phantom.theme.css
@import url(<"https://raw.githubusercontent.com/novimp3/phantom-theme/main/phantom.theme.css");
